1. Comprende el proyecto que vas a exponer

La Feria Rural Digital es un sitio web que simula una solucion para registrar visitantes interesados en productos, talleres y actividades comunitarias. Usa una base local simulada con JSON Server.

Tu trabajo consiste en dejarlo listo para una exposicion tecnica: comprobar que funcione, registrar evidencias, corregir detalles y explicar tus decisiones.

Frontend

HTML, CSS, JavaScript y Vite.

Datos

JSON Server como base simulada.

Interfaz

Bootstrap, iconos y componentes responsive.

Analisis

Chart.js para visualizar resultados.

4. Modo exposicion

Usa esta seccion como entrenamiento para tu presentacion final. No memorices todo: comprende cada punto y prepara evidencias.

Presenta el nombre del proyecto, el problema que resuelve y para quien fue creado.

Muestra el formulario, guarda un registro, revisa la tabla y abre JSON Server para demostrar que los datos existen.

Explica por que usaste Vite, JSON Server, Bootstrap, SweetAlert2 y Chart.js. Relaciona cada herramienta con una necesidad del proyecto.

Presenta al menos tres mejoras futuras con prioridad, razon tecnica y posible forma de implementacion.
