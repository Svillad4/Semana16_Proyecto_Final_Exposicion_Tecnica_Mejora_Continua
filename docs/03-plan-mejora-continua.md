## Ideas de mejora que puedes considerar

- Agregar inicio de sesion.
- Exportar registros a CSV.
- Mejorar accesibilidad.
- Agregar filtros de busqueda.
- Migrar la base de datos a Firebase o Supabase.
- Crear roles de usuario.
- Agregar panel administrativo.
- Mejorar rendimiento de carga.
- Agregar pruebas automatizadas.
- Mejorar documentacion tecnica.
