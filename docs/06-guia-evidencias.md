# 06 - Guia de evidencias

Las evidencias demuestran que realmente ejecutaste, probaste y preparaste tu proyecto.

## Evidencias obligatorias

| Carpeta | Evidencia esperada |
|---|---|
| `evidencias/proyecto-funcionando` | Captura del sitio abierto en navegador. |
| `evidencias/consola` | Captura de la consola sin errores criticos. |
| `evidencias/json-server` | Captura de `http://localhost:3001/visitantes`. |
| `evidencias/responsive` | Captura del sitio en vista movil. |
| `evidencias/pruebas` | Capturas de validaciones, formulario y registros. |
| `evidencias/exposicion` | Capturas, guion o material de apoyo para la exposicion. |

## Recomendaciones

1. Nombra tus capturas de forma clara.
2. No entregues capturas borrosas.
3. Incluye capturas del antes y despues si corregiste algo.
4. Verifica que las capturas muestren la URL o el contexto.
5. No dejes las carpetas vacias.

## Ejemplos de nombres de archivo

```text
01-proyecto-funcionando.png
02-consola-sin-errores.png
03-json-server-visitantes.png
04-formulario-validacion.png
05-vista-responsive.png
06-dashboard.png
```
