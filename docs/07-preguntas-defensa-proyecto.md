# 07 - Preguntas para defensa del proyecto

Prepara respuestas claras para estas preguntas. Te ayudaran a defender tu proyecto durante la exposicion.

1. Que problema resuelve tu proyecto?
2. Para que tipo de usuarios fue pensado?
3. Cual es la funcionalidad mas importante?
4. Por que usaste Vite?
5. Por que usaste JSON Server?
6. Por que usaste Bootstrap?
7. Como validas los datos del formulario?
8. Como sabes que los datos se guardan correctamente?
9. Que evidencias demuestran que probaste el proyecto?
10. Que error fue el mas dificil de corregir?
11. Que decision tecnica consideras mas importante?
12. Como organizaste los archivos del proyecto?
13. Como mejorarias la accesibilidad del sitio?
14. Que harias si tuvieras que convertir este proyecto en una aplicacion real?
15. Que aprendiste sobre mejora continua?
