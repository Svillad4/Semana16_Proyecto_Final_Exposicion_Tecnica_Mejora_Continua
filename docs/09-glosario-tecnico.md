# 09 - Glosario tecnico

| Concepto | Significado |
|---|---|
| Frontend | Parte visual e interactiva de una aplicacion web. |
| Backend | Parte que procesa datos y conecta con bases de datos. |
| JSON Server | Herramienta que simula una API usando un archivo JSON. |
| API | Interfaz que permite consultar o modificar datos desde una aplicacion. |
| Vite | Herramienta para ejecutar y construir proyectos web modernos. |
| Bootstrap | Libreria CSS para crear interfaces responsive. |
| SweetAlert2 | Libreria para mostrar alertas visuales modernas. |
| Chart.js | Libreria para crear graficos en JavaScript. |
| Responsive | Capacidad de una pagina para adaptarse a diferentes tamanos de pantalla. |
| Validacion | Revision de datos antes de procesarlos o guardarlos. |
| Evidencia | Captura o archivo que demuestra que una prueba fue realizada. |
| Argumentacion tecnica | Explicacion de por que se eligio una herramienta, estructura o solucion. |
| Mejora continua | Proceso de identificar y proponer nuevas mejoras para un proyecto. |
| README | Archivo que explica que es un proyecto, como instalarlo y como ejecutarlo. |
